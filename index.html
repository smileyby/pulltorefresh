API 基本配置项如下
mainElement: 下拉刷新的元素支持 id class tag
iconArrow: 下拉和释放时显示的图标
iconRefreshing: 刷新是显示的图标
instructionsPullToRefresh: 下拉时显示的文本
instructionsReleaseToRefresh: 释放时显示的文本
instructionsRefreshing: 刷新时显示的文本
refreshTimeout: 触发刷新回调函数的延迟时间（单位-毫秒）
onInit: 初始化回调函数
onRefresh: 触发刷新时的回调函数
